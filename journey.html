As I explored the topic, I learned the essentials of three Python libraries: NumPy, pandas and Matplotlib. Then, I did a few really basic projects related to the topic. You can find one here.


Getting to know what I like and don't like in other areas

I decided after some time to learn java, c, and C++ . Because of my interest in electrical engineering, C was the programming language I used most of the time. While learning Java and C++, I only made very simple console applications to practice the languages.
Following this, I began to learn the basics of web development. It consisted of HTML, CSS, and Javascript. Also, I have dedicated time to understanding how to work with bootstrap.
Following that, I began to devote myself to learning react(free and open-source JavaScript library for building user interfaces or UI components) as well as react native(mobile application framework which uses react with the native platform capabilities)
Eventually, at the beginning of 2021, I installed kali linux in a virtual machine for the first time. With the help of an online course, I learned about Nmap, Metasploit, and Wireshark.
To learn more about Metasploit, I installed the vulnerable operating system of Metaesploidable in a virtual Box. I got bored after a short time, as my interest in this area faded away, as well as my computer malfunctioning. So I decided to build a Personal portefolio using an HTML template found in this site. With time, I changed the website to the one you see now. Here is a link to the old site code. can be found here.
